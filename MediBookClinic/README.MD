### ✨ Основні можливості

- Герой-секція з потужним закликом до дії
- Послуги (Сімейний лікар, Кардіолог, Педіатр, Лабораторія)
- Чому обирають нас (6 карток переваг)
- Як записатися (4 простих кроки)
- **Наші лікарі** — окрема секція з фото, спеціальністю та досвідом
- Галерея клініки та щасливих пацієнтів
- Карусель відгуків
- FAQ у вигляді акордеону
- Форма онлайн-запису з повідомленнями про успіх/помилку
- Контактна панель + плаваюча кнопка WhatsApp
- Футер з прямим посиланням на GitHub
- Прелодер, плавна прокрутка, анімації AOS
