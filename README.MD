# 🚀 Your Site PRO v1.0 — Універсальний Landing Page + Адмінка

**Сучасний односторінковий сайт будь-якої тематики з потужною адмін-панеллю.**  
Без бази даних. Без фреймворків. Повністю керований через браузер.

**Ідеально підходить для:** клінінгу, автомийки, доставки, стоматології, салонів краси, ремонту, фотографів, юристів, ресторанів — просто змінюйте тексти та зображення.

## ✨ Головні можливості (v1.0)

### 🎛️ Адмін-панель (admin.php)
- **Захищений вхід** паролем (змінюється за 5 секунд)
- **14 зручних вкладок**:
  - Загальні налаштування
  - **Шапка сайту** ← логотип + повне меню (додавання/видалення пунктів)
  - Meta Tags (title, description, keywords, robots, author, theme-color, canonical)
  - Open Graph + Twitter Cards
  - Hero секція (H1, текст, кнопки)
  - Послуги (зображення, заголовок, опис, футер)
  - Переваги (emoji + текст)
  - Як ми працюємо (кроки)
  - Галерея (фото + alt)
  - Відгуки (зірки, текст, ім’я, дата)
  - SEO текст (H2 + два параграфи)
  - FAQ (питання/відповіді)
  - Форма замовлення (успіх/помилка + лейбли)
  - Footer + Schema.org
  - **Custom Code** (head, CSS, JS, HTML перед `</body>`)

- Живе редагування в реальному часі
- Автоматичне збереження в `content.json` з красивим форматуванням
- Мобільна адаптивна панель + бокове меню
- TailwindCSS + AlpineJS — швидкий і красивий інтерфейс 2026 року

### 🌐 Головний сайт (index.php)
- 100% динамічний — всі дані з `content.json`
- Повністю адаптивний (мобільний + десктоп)
- Максимальна SEO-оптимізація (meta, OG, Twitter, Schema.org)
- Форма замовлення з обробкою
- Контактний бар (телефон, WhatsApp, Viber, Telegram)
- Готова підтримка багатомовності (ua.php, ru.php, en.php тощо)
- Сучасні анімації, hover-ефекти, швидке завантаження

## 🚀 Як встановити (3 хвилини)

1. Завантажте всі файли на будь-який PHP-хостинг
2. Відкрийте `admin.php`
3. Введіть пароль `admin123` (змініть у першому рядку!)
4. Редагуйте будь-який текст, меню, фото — натискайте **«ЗБЕРЕГТИ ВСІ ЗМІНИ»**
5. Відкрийте `index.php` — ваш сайт вже живий!

Готово. Ніяких баз даних, ніяких складнощів.
