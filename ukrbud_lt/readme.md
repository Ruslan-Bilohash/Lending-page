# 🧹 AI Consultant для Ukrbud.lt (Baltic Clean)

**Інтелектуальний AI-чатбот для бронювання професійного прибирання у Вільнюсі та околицях**

Сучасний AI-консультант на базі Grok xAI, який автоматично відкривається на сайті, першим вітає клієнта та активно допомагає забронювати прибирання.

---

### ✨ Що робить цей бот

- **Автоматично відкривається** через 4 секунди після завантаження сторінки
- **Говорить першим** — відразу пропонує забронювати прибирання
- Підтримує **4 мови**: литовську (основна), російську, українську та англійську
- Автоматично визначає мову клієнта і відповідає тією самою мовою
- Активно кваліфікує заявку (тип прибирання, площа, район, терміни)
- Збирає контакти клієнта (ім’я + телефон)
- Пропонує розрахунок ціни та переваги компанії
- Переводить розмову на бронювання замовлення
- Всі повідомлення дублюються в Telegram адміністратору
- Повна історія розмов зберігається на сервері

### 🛠 Технології

- **Backend**: PHP 8 + Grok xAI API (`grok-4.20-0309-non-reasoning`)
- **Frontend**: Чистий JavaScript (без фреймворків)
- **Зберігання**: JSON-файли (історія розмов)
- **Сповіщення**: Telegram Bot API
- **Дизайн**: Неоновий стиль з backdrop-filter та сучасними градієнтами
