### 🧹 **Vilnius Cleaning 2026** — Преміум мультимовний сайт професійного клінінгу

**Повністю готовий преміум-лендінг 2026 року** для компанії професійного прибирання у Вільнюсі (Литва). 5 мов, максимальне SEO, красивий дизайн, сучасна форма замовлення та професійний HTML-лист.

---

### ✨ **Головні переваги проекту**

- **5 повноцінних мовних версій** (всі файли окремі та повністю перекладені):
  - 🇱🇹 Литовська — `index.php` (головна)
  - 🇬🇧 Англійська — `en.php`
  - 🇺🇦 Українська — `ua.php`
  - 🇷🇺 Російська — `ru.php`
  - 🇳🇴 Норвезька — `no.php`

- **100% тематика клінінгу** — всі фото професійні (квартири, офіси, після ремонту, пилососи, ганчірки, швабри, еко-засоби, чисті інтер’єри)

- **Максимальне SEO 2026**:
  - Розширені meta-теги + keywords
  - Open Graph + Twitter Cards
  - Schema.org (LocalBusiness)
  - Canonical, robots, theme-color
  - Довгі SEO-тексти + LSI-ключі
  - Оптимізовані заголовки

- **Красива форма замовлення** з валідацією та перенаправленням після відправки
- **Сучасний HTML-лист** на пошту (gradient, стильний дизайн, кнопки, логотип)
- **Стильна сторінка 404** з таймером автопереадресації на головну
- **Автоматична обробка 404** через `.htaccess`
