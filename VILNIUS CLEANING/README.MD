  - 🇳🇴 Норвезька — [`no.php`](https://meistru.lt/no.php)

- **Нові потужні файли v2.0**:
  - `kalkuliatorius.php` — сучасний калькулятор з реальним розрахунком, автоматичною 30% знижкою, таймером 10 хвилин (localStorage + cookie), валідацією та красивим HTML-листом
  - `traffic_logger.php` — розумний логер з визначенням країни, міста, джерела трафіку (Google, TikTok, Facebook, Instagram, YouTube), пристрою та кешуванням IP
  - `sitemap.xml` — повністю оновлений динамічний з пріоритетами та датами
  - `robots.txt` — оптимізований для багатосторінкового сайту з 5 мовами
  - `.htaccess` — покращена обробка 404, параметрів lang/success, захист папок
  - `404.php` — стильна сторінка 404 з автоматичним таймером перенаправлення на головну (8 секунд)
  - `donate.php` — окрема сторінка підтримки автора (Buy Me a Coffee, Wise, PayPal, Vipps + QR-код)
  - `about.html` — повна документація проекту з вкладками для всіх мов
  - `ifile.php` — зручний файловий менеджер (бонус для адміністрування)

- **100% тематика клінінгу** — професійні фото (квартири, офіси, після ремонту, пилососи, ганчірки, швабри, еко-засоби, чисті інтер’єри)

  - Розширені meta-теги + keywords + LSI
  - Open Graph + Twitter Cards

  - Canonical, robots, theme-color
  - Довгі SEO-тексти 400+ слів у кожній мовній версії
  - Оптимізовані заголовки H1–H3

- **Красива форма замовлення** з валідацією, CSRF-захистом та автоматичним перенаправленням
- **Сучасний HTML-лист** на пошту (gradient, таблиця даних, кнопки, логотип) — приходить на 3 адреси
