- **Повністю адаптивний** дизайн на Tailwind CSS

## 🚀 Як запустити

1. Завантаж всі файли у корінь сайту (`public_html` або `www`)
2. Налаштуй пошту у файлі `submit.php` (рядок `$to`)
3. Зміни пароль у `ifile.php` (якщо використовуєш)
4. Завантаж `robots.txt` і `sitemap.xml`
5. Додай сайт у Google Search Console + подай sitemap
