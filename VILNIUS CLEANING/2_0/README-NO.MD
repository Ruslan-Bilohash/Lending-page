# 🚀 Your Site PRO v1.0 — Universell Landingsside + Adminpanel
**Moderne énside-nettside for enhver nisje med kraftig adminpanel.**
Ingen database. Ingen rammeverk. Fullstendig styrt via nettleseren.
**Perfekt for:** renhold, bilvask, levering, tannlege, skjønnhetssalonger, reparasjon, fotografer, advokater, restauranter — bare endre tekster og bilder.

## ✨ Hovedfunksjoner (v1.0)
### 🎛️ Adminpanel (admin.php)
- **Passordbeskyttet innlogging** (endres på 5 sekunder)
- **14 praktiske faner**:
  - Generelle innstillinger
  - **Nettstedets topp** ← logo + full meny (legge til/fjerne punkter)
  - Meta Tags (title, description, keywords, robots, author, theme-color, canonical)
  - Open Graph + Twitter Cards
  - Hero-seksjon (H1, tekst, knapper)
  - Tjenester (bilder, tittel, beskrivelse, footer)
  - Fordeler (emoji + tekst)
  - Hvordan vi jobber (trinn)
  - Galleri (bilder + alt)
  - Anmeldelser (stjerner, tekst, navn, dato)
  - SEO-tekst (H2 + to avsnitt)
  - FAQ (spørsmål/svar)
  - Bestillingsskjema (suksess/feil + etiketter)
  - Footer + Schema.org
  - **Custom Code** (head, CSS, JS, HTML før `</body>`)
...
