# 🧹 Vilnius Cleaning 2026 — Premium flerspråklig profesjonell renholdsside
Fullstendig klar premium landingsside 2026 for profesjonelt renholdsfirma i Vilnius (Litauen). 5 språk, maksimal SEO, vakkert design, moderne bestillingsskjema og profesjonell HTML-e-post.
✨ Hovedfordeler med prosjektet

* 5 fullverdige språkversjoner (alle filer er separate og fullstendig oversatt):

  * 🇱🇹 Litauisk — index.php (hoved)

  * 🇬🇧 Engelsk — en.php

  * 🇺🇦 Ukrainsk — ua.php

  * 🇷🇺 Russisk — ru.php

  * 🇳🇴 Norsk — no.php

* 100% renholdstema — alle bilder er profesjonelle (leiligheter, kontorer, etter renovering, støvsugere, kluter, mopper, eco-midler, rene interiører)

* Maksimal SEO 2026:

  * Utvidet meta-tagger + keywords

  * Open Graph + Twitter Cards

  * Schema.org (LocalBusiness)

  * Canonical, robots, theme-color

  * Lange SEO-tekster + LSI-nøkkelord

  * Optimaliserte overskrifter

* Vakkert bestillingsskjema med validering og omdirigering etter innsending

* Moderne HTML-e-post (gradient, stilig design, knapper, logo)

* Stilfull 404-side med autotimer for omdirigering til hovedsiden

* Automatisk 404-håndtering via .htaccess

📱 Design og UX

* Moderne Tailwind CSS + jevne animasjoner

* Hero-seksjon fullskjerm

* «Før og Etter» galleri

* Fordelsblokk med ikoner

* 4-trinns arbeidsprosess

* Kundeanmeldelser

* FAQ (trekkspill)

* Kontaktlinje (telefon + WhatsApp + Viber + Telegram)

* Mobilmeny + språkdropdown med flagg

📧 Skjema og e-post

* submit.php — pålitelig behandler

* Vakkert profesjonelt HTML-brev (med gradient, datatabell og lenke til siden)

* Automatisk omdirigering til samme språkversjon etter innsending

* Beskyttelse mot tomme felt

🛠 Teknisk stack

* PHP 8+

* Tailwind CSS

* Font Awesome 6

* Google Fonts (Inter + Playfair Display)

* Schema.org JSON-LD

* .htaccess for SEO og 404
