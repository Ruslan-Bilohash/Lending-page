# 🧹 Vilnius Cleaning 2026 — Professional Multi-language Cleaning Website

**Fully completed premium multi-language landing page** for a professional cleaning company in Vilnius, Lithuania.

### ✨ Main Features

- **5 full language versions**:
  - 🇱🇹 Lithuanian (`index.php`)
  - 🇬🇧 English (`en.php`)
  - 🇺🇦 Ukrainian (`ua.php`)
  - 🇷🇺 Russian (`ru.php`)
  - 🇳🇴 Norwegian (`no.php`)

- **100% cleaning theme** — all images replaced with professional cleaning photos (apartments, offices, post-renovation, vacuums, cloths, mops, eco-products, sparkling interiors). No unrelated photos.

- **Maximum SEO 2026**:
  - Extended meta tags
  - Open Graph + Twitter Cards
  - Schema.org LocalBusiness
  - Canonical, robots, LSI keywords
  - Long SEO texts 400+ words

- Beautiful order form with stylish HTML email
- Modern 404 page with auto-redirect to home (8 seconds)
- Fully responsive + mobile menu
- Fast loading and clean code

3. Upload `.htaccess`
